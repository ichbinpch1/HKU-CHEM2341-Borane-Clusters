Wade’s rules (quick reference)

Relate number of vertices \(n\) to skeletal electron pairs.

Classes

closo
Skeletal e− pairs = n + 1
nido
Skeletal e− pairs = n + 2
arachno
Skeletal e− pairs = n + 3

Counting reminders

• For closo boranes like [BnHn]2−, a common shortcut is “BH = 2 e” + charge.
• For neutral nido boranes like B5H9, use total valence e− then subtract 2 e for each terminal B–H bond to get skeletal e−.
